# feffery-antd-components 文档索引（拆分版）

> 本目录基于 `feffery-antd-docs` 仓库源码自动汇总，面向离线阅读与 RAG。

## 目录结构

- `01-guides.md`：指南与概览（what_is_fac / getting_started / version_migration_guide）。
- `02-advanced-usage.md`：高级用法（views/advanced_usage）。
- `components/`：每个组件一个 markdown，包含简介源码、示例源码（demos）、API 文档与补充说明。
- `api-only/`：仅有 API 文档、无对应视图源码的组件。
